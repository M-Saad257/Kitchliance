# Kitchliance 🍳

## Overview

Kitchliance is a modern kitchen appliance startup website built to showcase and sell kitchen products such as air fryers, appliances, and home kitchen tools. The platform is designed with a clean modern UI and focuses on performance, simplicity, and a smooth shopping experience.

The goal of Kitchliance is to create a trusted online destination for high‑quality kitchen appliances with an attractive interface and scalable web architecture.

---

## 🚀 Tech Stack

**Frontend**

* React
* Vite
* Tailwind CSS
* React Router

**Design Approach**

* Modern responsive UI
* Hero section with appliance showcase
* Product cards with images
* Clean navigation with icon interactions

---

## ✨ Features

### 1. Responsive Navbar

* Fixed top navigation
* Smooth hover effects
* Icons inside navigation links
* Works across mobile and desktop

### 2. Hero Section

* Large background appliance image
* Marketing headline and CTA
* Layer control with navbar using z-index

### 3. Product Showcase

* Appliance cards displaying:

  * Product image
  * Product name
  * Basic description
  * Price

### 4. Product Images

* High quality appliance images
* Full image display without cropping
* Focus on realistic product visuals

### 5. Clean UI

* Modern color palette
* Kitchen brand focused design
* Consistent spacing and layout
